# Guide de contribution

Merci de votre intérêt pour contribuer à GithubWrapped !

## Développement local

1. Fork le repository
2. Clone votre fork : `git clone https://github.com/votre-username/GithubWrapped.git`
3. Créez une branche : `git checkout -b feature/ma-fonctionnalite`
4. Installez les dépendances : `npm install`
5. Lancez le serveur de développement : `npm run dev`
6. Faites vos modifications
7. Testez vos changements
8. Committez : `git commit -m "Ajout de ma fonctionnalité"`
9. Push : `git push origin feature/ma-fonctionnalite`
10. Ouvrez une Pull Request

## Standards de code

- Utilisez TypeScript pour tous les nouveaux fichiers
- Suivez les conventions de nommage existantes
- Ajoutez des commentaires pour le code complexe
- Testez vos modifications avant de soumettre une PR

## Structure du projet

- `src/components/` : Composants React réutilisables
- `src/services/` : Services API et logique métier
- `src/hooks/` : Hooks React personnalisés
- `src/utils/` : Fonctions utilitaires
- `src/types/` : Définitions TypeScript

## Questions ?

Ouvrez une issue pour toute question ou suggestion !
